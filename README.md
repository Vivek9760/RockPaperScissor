# 🪨📄✂️ Rock Paper Scissors — Totally Fair Game™

🌐 **Live Site:** [Play Now](https://vivek9760.github.io/RockPaperScissor/)

---

A modern, interactive Rock Paper Scissors game built with vanilla HTML, CSS, and JavaScript. Clean UI, smooth animations, and an AI opponent that is *probably* fair.

---

## ✨ Features

- Interactive gameplay with instant results  
- AI opponent with randomized choices  
- Smooth animations (pop, shake, transitions)  
- Live scoreboard (You / AI / Draw)  
- Sarcastic result messages for every outcome  
- Responsive design for all screen sizes  
- Reset functionality to restart the game  

---

## 🛠 Tech Stack

- HTML5, CSS3, Vanilla JavaScript  
- Custom animations using CSS keyframes  
- No frameworks, no dependencies
